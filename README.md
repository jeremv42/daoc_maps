# DAOC Maps

A simple project to see all Dark Age of Camelot maps in a web page. With the possibility to add markers and publish its map with its own point of interest.

## Project setup
```
npm ci
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
